# 2D Survival Game Assets

This repository contains all pixel art assets for the project.

# Folder Structure

- **Environment/**
  - Tilesets, map objects, props, decorations
- **Equipments/**
  - Weapons, armors, tools, items
- **Projectiles/**
  - Bullets, arrows, magic projectiles, VFX
- **Skins/**
  - Character skins, outfits, costumes
- **UI_UX/**
  - Interface icons, menus, buttons, HUD elements
